# Custom Items Mod

A Fabric mod for Minecraft 1.21.4 that adds custom tools to the game.

## Features
- **Ruby Tools**: Diamond-tier custom tools with enhanced properties
  - Ruby Pickaxe
  - Ruby Shovel
  - Ruby Axe
  - Ruby Sword
- **Emerald Tools**: Netherite-tier custom tools
  - Emerald Pickaxe
  - Emerald Shovel

## Installation
1. Download Fabric Loader for Minecraft 1.21.4
2. Place this mod in your mods folder
3. Launch Minecraft with the Fabric profile

## Building from Source
`./gradlew build`

The compiled JAR will be in build/libs/

## Requirements
- Java 21+
- Fabric Loader 0.15.0+
- Minecraft 1.21.4
- Fabric API 0.92.2+
